🍝 Аналіз продажів італійського ресторану
📊 Опис проєкту

Цей проєкт присвячений аналізу даних продажів італійського ресторану.
Аналіз виконано за допомогою Python, а результати представлені у вигляді інтерактивного дашборду в Microsoft Power BI.

Мета проєкту — дослідити структуру продажів, визначити найбільш прибуткові категорії та страви, знайти пікові години роботи ресторану та отримати бізнес-інсайти на основі даних.

🎯 Цілі аналізу

дослідити структуру продажів ресторану

визначити найбільш прибуткові категорії страв

знайти топ-страви за доходом

визначити пікові години та дні продажів

створити інтерактивний дашборд для візуалізації результатів
